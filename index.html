Computer Science Abstractions

Overview of the data structures and algorithms mapped between the C++ Engine and the Web Portal.

Contiguous Store (std::vector)

C++ Mapping: std::vector<Student> students
JS Mapping: studentsCache Array

Maintains Contiguous memory blocks for student lists, enabling $O(1)$ index access. Serves as the primary source array for sorting, rendering directories, and executing queries.

Hash Index Map (std::unordered_map)

C++ Mapping: std::unordered_map<int,int> rollIndex
JS Mapping: rollIndex key-value Hash Object

Maps roll numbers to array indexes. Translates search operations from $O(N)$ linear scans to $O(1)$ average time lookups. Critical for verifying registration duplicates during additions.

FIFO Session Queue (std::queue)

C++ Mapping: std::queue<int> q
JS Mapping: attendanceQueue with Array Shift operations

Processes batches of students in First-In-First-Out order during marking. Ensures students are evaluated in the precise sequence of roll numbers entered by the instructor.

LIFO Undo Stack (std::stack)

C++ Mapping: std::stack<std::pair<int,string>> undoStack
JS Mapping: attendanceStack with Array Push/Pop

Records attendance marks chronologically. Enables LIFO (Last-In-First-Out) undo. Popping from the stack decrements student record increments and shifts elements back into the active marking queue.

Binary Search (O(log N) Lookup)

Prerequisite: Sorted dataset by key (Roll No)

Halves the search space at each comparison step. Runs in logarithmic time. Unsorted datasets dynamically trigger pre-sorting before execution, comparing $O(N)$ scanning with $O(\log N)$ division.

Sorting Visualizer (Bubble vs Merge)

Bubble Sort: $O(N^2)$ (In-place, stable swaps)
Merge Sort: $O(N \log N)$ (Recursive Divide & Merge)

Compares Bubble Sort's quadratic scaling against Merge Sort's logarithmic recursively split sorting. Visualization color-codes elements during comparison steps, swaps, and sorted alignments.
