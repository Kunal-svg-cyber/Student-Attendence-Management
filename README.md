# Student Attendance Management System

A C++ Console Application utilizing Data Structures and Algorithms (DSA) to manage student attendance.

## Project Structure
- `Student.h` / `Student.cpp`: Defines the `Student` entity (Roll No, Name, Department, Attendance records).
- `AttendanceSystem.h` / `AttendanceSystem.cpp`: Manages the collection of students, add/delete operations, queue-based marking, and stack-based undo.
- `FileManager.h`: Handles reading and writing student data to `students.txt`.
- `Search.h`: Implements Linear Search and Binary Search algorithms.
- `MergeSort.h`: Implements Bubble Sort and Merge Sort algorithms.
- `main.cpp`: Entry point containing the console-based menu interface.
- `students.txt`: Plain text database file for persisting student records.
